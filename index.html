Windows PowerShell Scripting

A practical, beginner-friendly guide to creating, executing, and enhancing PowerShell scripts — with modern tools that make Windows configuration faster and smarter.

1. Creating a PowerShell Script

PowerShell scripts are plain text files with a .ps1 extension. They allow you to automate tasks such as system configuration, software installation, and maintenance.

  1. Open Notepad or Visual Studio Code.
  2. Write your PowerShell commands.
  3. Save the file with a .ps1 extension (e.g. setup.ps1).
# Example PowerShell Script Write-Host "Hello from PowerShell!" Get-Date

2. Allowing Script Execution

By default, Windows restricts script execution for security reasons. You may need to adjust the execution policy before running your script.

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
Always understand what a script does before running it, especially if it was downloaded from the internet.

3. Running the Script

Once your script is ready, you can execute it directly from PowerShell.

cd C:\Path\To\Your\Script .\setup.ps1

4. Using the Chris Titus Tech Windows Utility

The utility can be launched directly from PowerShell using the following command:
